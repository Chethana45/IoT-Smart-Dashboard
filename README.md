# ⚡ Smart IoT Monitoring Dashboard

A modern and interactive IoT Dashboard built using **HTML, CSS, JavaScript, and Node.js**.

This project simulates real-time environmental monitoring with dynamic UI components, device control, and live data visualization.

---

## 🚀 Features

- 🌡 Real-time Temperature Simulation
- 💧 Humidity Monitoring
- 💡 Smart Device ON/OFF Toggle
- 📊 Live Animated Sensor Chart (Canvas API)
- 🌙 Dark / Light Mode Toggle
- 📱 Fully Responsive Design
- 🌐 Node.js Static Server Integration

---

## 🛠 Technologies Used

- HTML5
- CSS3 (Glassmorphism UI)
- JavaScript (DOM + Canvas API)
- Node.js (HTTP Server)
